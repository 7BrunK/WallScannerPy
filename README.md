# WallScannerPy

**Мой личный сканер для автоматического сохранения учебных слайдов презентаций (для дискреточки).**

WallScannerPy — это кроссплатформенное приложение с графическим интерфейсом, использующее компьютерное зрение для автоматического обнаружения и сохранения слайдов с проектора. Программа отслеживает область кадра, фиксирует проекционный экран/стену и автоматически сохраняет новый слайд при его изменении, избавляя от необходимости делать снимки вручную.

## 🚀 Основные возможности

*   **Автоматическое обнаружение проекционной области**: Автоматически находит и выделяет область проецируемого слайда в кадре.
*   **Отслеживание изменений**: Сравнивает текущий и предыдущий кадры проекционной области. При обнаружении существенных изменений (смены слайда) автоматически делает "снимок".
*   **Автосохранение**: Все новые слайды сохраняются в указанную директорию с автоматической нумерацией.
*   **Предпросмотр в реальном времени**: Окно приложения отображает видеопоток с наложенной рамкой, выделяющей найденную проекционную область.
*   **Ручное управление**: Возможность принудительной остановки автоматического режима.

## 🛠 Технологии и библиотеки

Проект построен на следующих технологиях:

*   **[Kivy](https://kivy.org/)** — фреймворк для создания графического интерфейса приложения.
*   **[OpenCV-Python](https://opencv.org/)** — основная библиотека для обработки изображений, обнаружения контуров и сравнения кадров.
*   **[NumPy](https://numpy.org/)** — для эффективной работы с многомерными массивами (кадрами) и выполнения математических операций.

## 🧪 Как это работает

1.  **Поиск области проекции**: Каждый кадр с веб-камеры преобразуется в оттенки серого и обрабатывается для поиска крупных четырехугольных контуров (вероятная область экрана/стены).
2.  **Стабилизация области**: Найденная область выпрямляется, чтобы получить "фронтальный" вид слайда.
3.  **Сравнение кадров**: Выпрямленное изображение текущего слайда сравнивается с сохраненным эталоном предыдущего слайда. *Сейчас используется примитивная метрика различия.
4.  **Сохранение**: Текущий выпрямленный слайд сохраняется как PNG-файл в выбранную папку, и эталон обновляется.

## 💻 Установка и запуск

1.  **Клонируйте репозиторий**:
    ```bash
    git clone https://github.com/7BrunK/WallScannerPy.git
    cd WallScannerPy
    ```

2.  **Установите зависимости**:
    ```bash
    pip install -r requirements.txt
    ```

3.  **Запустите приложение**:
    ```bash
    python main.py
    ```
